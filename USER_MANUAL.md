## 시작하기

### 시스템 요구사항

- **웹 브라우저**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **인터넷 연결**: 안정적인 인터넷 연결 필요
- **화면 해상도**: 최소 1024x768, 권장 1920x1080 이상

### 접속 방법

1. **웹 브라우저에서 접속**
   - 주소: `https://insitechart.com`
   - 또는 로컬 설치 시: `http://localhost:8501`

2. **계정 생성 (선택사항)**
   - 우측 상단의 "로그인/회원가입" 클릭
   - 필요 정보 입력 후 계정 생성

### 첫 화면 소개

메인 대시보드는 다음 섹션으로 구성됩니다:

- **📊 대시보드**: 전체 시장 현황과 주요 지표
- **🔍 주식 분석**: 개별 주식 상세 분석
- **📈 주식 비교**: 여러 주식 비교 분석
- **💬 감성 분석**: 소셜 미디어 감성 분석
- **🌊 시장 개요**: 시장 전체 현황
- **⚙️ 설정**: 개인화 설정

## 주요 기능

### 실시간 데이터
- **주가 정보**: 실시간 주가, 거래량, 변동률
- **감성 데이터**: 소셜 미디어 실시간 감성 분석
- **시장 지수**: 주요 시장 지수 현황

### 고급 차트
- **다양한 차트 타입**: 캔들스틱, 라인, 영역 차트
- **기술적 지표**: RSI, MACD, 볼린저 밴드, 이동평균선
- **사용자 정의**: 기간 설정, 지표 조합

### 감성 분석
- **다중 소스**: Reddit, Twitter, Discord 데이터
- **실시간 분석**: AI 기반 감성 점수 계산
- **트렌드 식별**: 인기 주식 및 트렌드 파악

## 대시보드 사용법

### 메인 대시보드

1. **시장 감성 개요**
   - 전체 시장 감성 상태를 게이지로 표시
   - 긍정/부정/중립 비율 확인

2. **인기 주식**
   - 현재 트렌딩 중인 주식 목록
   - 트렌드 점수와 감성 정보 표시

3. **시장 지수**
   - S&P 500, 다우존스, 나스닥 등 주요 지수
   - 실시간 변동률 표시

4. **빠른 통계**
   - 선택한 주식의 주요 지표
   - 현재가, 거래량, 시가총액 등

### 워치리스트 관리

1. **주식 추가**
   - 좌측 패널에서 주식 심볼 입력
   - "➕ Add to Watchlist" 버튼 클릭

2. **워치리스트 확인**
   - 추가된 주식 목록 표시
   - 클릭하여 상세 정보 이동

3. **주식 제거**
   - 제거할 주식 선택 후 "🗑️ Remove" 클릭

## 주식 분석

### 주식 검색

1. **기본 검색**
   - 상단 검색창에 주식명 또는 심볼 입력
   - 실시간 검색 결과 표시

2. **필터링**
   - 유형별 필터: 주식, ETF, 뮤추얼펀드, 지수
   - 거래소별 필터링 가능

### 상세 정보

1. **기본 정보**
   - 회사명, 심볼, 거래소
   - 섹터, 산업 분류

2. **재무 지표**
   - 현재가, 전일 종가, 변동률
   - 시가총액, P/E 비율, 배당수익률
   - 베타, 52주 최고/최저가

3. **감성 정보**
   - 종합 감성 점수 (게이지 표시)
   - 24시간/1시간 언급량
   - 긍정/부정/중립 언급 분포

### 차트 분석

1. **시간대 선택**
   - 1일, 1주, 1개월, 3개월, 6개월, 1년, 2년, 5년, 전체
   - 버튼 클릭으로 간편한 시간대 전환

2. **차트 타입**
   - **캔들스틱**: 시가/고가/저가/종가 표시
   - **라인**: 종가 연결 라인 차트
   - **영역**: 종가 아래 영역 채우기

3. **기술적 지표**
   - **볼린저 밴드**: 가격 변동성 밴드
   - **RSI**: 상대강도지수 (과매도/과매수 구간)
   - **MACD**: 추세 전환 지표
   - **이동평균선**: SMA, EMA 다양한 기간 지원

4. **거래량 분석**
   - 거래량 막대 차트
   - 거래량 이동평균선

## 감성 분석

### 시장 감성

1. **전체 감성**
   - 시장 전체의 감성 상태
   - 긍정/부정/중립 비율 파이 차트

2. **감성 분포**
   - 산업별 감성 분포
   - 시간대별 감성 변화

### 개별 주식 감성

1. **감성 점수**
   - -1에서 +1 사이의 감성 점수
   - 색상으로 긍정/부정/중립 구분

2. **언급 통계**
   - 24시간 언급량
   - 긍정/부정/중립 언급 수

3. **커뮤니티별 감성**
   - Reddit, Twitter, Discord별 감성
   - 각 플랫폼의 특성 반영

4. **실제 언급 내용**
   - "Include Mentions" 체크 시 실제 게시물 표시
   - 작성자, 내용, 감성 점수, 추천수

### 트렌딩 주식

1. **트렌드 식별**
   - 언급량 급증 주식
   - 감성 점수와 트렌드 점수

2. **트렌드 지속시간**
   - 트렌드 시작 시간
   - 지속 기간 표시

## 주식 비교

## 문제 해결

### 일반적인 문제

1. **데이터가 표시되지 않을 때**
   - 인터넷 연결 확인
   - 브라우저 새로고침 (F5)
   - API 상태 확인

2. **차트가 로드되지 않을 때**
   - 브라우저 캐시 삭제
   - 다른 브라우저에서 시도
   - 시간대 변경 후 다시 시도

3. **감성 데이터가 없을 때**
   - 일부 주식은 감성 데이터 제공 안됨
   - 다른 주식으로 시도

### 성능 문제

1. **로딩이 느릴 때**
   - 불필요한 탭 닫기
   - 차트 시간대 단축
   - 브라우저 확장 프로그램 비활성화

2. **메모리 사용량이 높을 때**
   - 브라우저 재시작
   - 다른 탭 닫기
   - 저사양 모드 사용 (설정에서)

### 오류 메시지

1. **"API 연결 실패"**
   - 인터넷 연결 확인
   - 잠시 후 다시 시도
   - 고객지원 문의

2. **"데이터 로딩 오류"**
   - 새로고침 후 다시 시도
   - 다른 브라우저 사용
   - 캐시 삭제

## FAQ

### 일반 질문

**Q: 무료로 사용할 수 있나요?**
A: 기본 기능은 무료로 사용 가능합니다. 고급 기능은 유료 구독이 필요할 수 있습니다.

**Q: 어떤 주식 데이터를 제공하나요?**
A: 전 세계 주요 시장의 주식 데이터를 제공합니다. 국가별로 제공되는 종목이 다를 수 있습니다.

**Q: 감성 분석은 어떻게 작동하나요?**
A: Reddit, Twitter, Discord 등 소셜 미디어 데이터를 AI로 분석하여 투자자 심리를 파악합니다.

**Q: 데이터는 얼마나 자주 업데이트되나요?**
A: 주가 데이터는 실시간으로, 감성 데이터는 15분 간격으로 업데이트됩니다.

**Q: 모바일에서 사용할 수 있나요?**
A: 네, 반응형 웹 디자인으로 모바일에서도 최적화된 경험을 제공합니다.

### 기술적 질문

**Q: 어떤 브라우저를 지원하나요?**
A: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+를 지원합니다.

**Q: 데이터는 어떻게 보호되나요?**
A: SSL 암호화로 데이터 전송되며, 개인정보는 안전하게 저장됩니다.

**Q: API 제한이 있나요?**
A: 네, 공정한 사용을 위해 분당 요청 횟수가 제한됩니다.

### 고급 기능

**Q: API를 직접 사용할 수 있나요?**
A: 네, 개발자를 위한 API 문서를 제공합니다. 설정의 "API 문서"를 참조하세요.

**Q: 데이터를 내보낼 수 있나요?**
A: 네, CSV, JSON 형식으로 데이터 내보내기가 가능합니다.

**Q: 알림 기능이 있나요?**
A: 네, 가격 변동, 감성 변화 등 알림을 설정할 수 있습니다.
