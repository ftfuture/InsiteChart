# InsiteChart 프로젝트 분석 보고서

## 개요

InsiteChart는 실시간 주식 데이터와 소셜 미디어 감성 분석을 통합한 차세대 금융 분석 플랫폼입니다. 사용자들은 종합적인 시장 인사이트, 실시간 데이터 동기화, 및 고급 분석 도구를 통해 더 나은 투자 결정을 내릴 수 있습니다.

## 아키텍처

### 시스템 구조
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   프론트엔드     │    │   API 게이트웨이  │    │   마이크로서비스  │
│  (Streamlit)    │◄──►│   (FastAPI)     │◄──►│   (Python)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                 │                        │
                                 ▼                        ▼
                        ┌─────────────────┐    ┌─────────────────┐
                        │   캐시 시스템    │    │   데이터베이스    │
                        │   (Redis)      │    │  (PostgreSQL)   │
                        └─────────────────┘    └─────────────────┘
```

### 핵심 컴포넌트

- **백엔드 API**: FastAPI 기반의 RESTful API
- **프론트엔드**: Streamlit 기반의 웹 인터페이스
- **데이터 소스**: Yahoo Finance, Reddit, Twitter
- **캐싱**: Redis 기반의 다계층 캐싱
- **데이터베이스**: PostgreSQL 기반의 데이터 저장
- **모니터링**: Prometheus & Grafana

## 주요 기능

### 💰 금융 데이터
- 실시간 주식 가격 및 시장 데이터
- 포괄적인 재무 정보 및 지표
- 과거 데이터 분석 및 차트
- 시장 지수 모니터링

### 📊 소셜 감성 분석
- Reddit, Twitter, Discord에서 실시간 감성 데이터 수집
- AI 기반 감성 분석 및 트렌드 식별
- 커뮤니티별 감성 분석
- 트렌딩 주식 식별

### 🔄 통합 분석
- 주식 데이터와 감성 분석의 통합
- 상관관계 분석 및 비교 도구
- 시장 인사이트 및 예측
- 개인화된 워치리스트 및 알림

### 🌐 고급 기능
- 다국어 지원 (한국어, 영어, 일본어 등 10개 언어)
- 실시간 데이터 동기화 (WebSocket)
- 고급 캐싱 시스템
- API 레이트리밋 및 보안
- 모니터링 및 알림 시스템

## 기술 스택

### 백엔드
- **프레임워크**: FastAPI
- **데이터베이스**: PostgreSQL, Redis
- **데이터 처리**: pandas, numpy
- **감성 분석**: VADER, NLTK, TextBlob
- **외부 API**: Yahoo Finance, Reddit, Twitter
- **인증**: JWT
- **모니터링**: Prometheus, Grafana

### 프론트엔드
- **프레임워크**: Streamlit
- **데이터 시각화**: Plotly
- **차트 라이브러리**: ta (Technical Analysis)

### 인프라
- **컨테이너화**: Docker, Docker Compose
- **CI/CD**: GitHub Actions
- **데이터베이스**: PostgreSQL, Redis
- **리버스 프록시**: Nginx
- **모니터링**: Prometheus, Grafana

```
│   ├── api/                   # API 라우트

│   ├── middleware/             # 미들웨어
│   ├── monitoring/            # 모니터링
│   ├── services/             # 비즈니스 로직
│   ├── models/               # 데이터 모델
│   └── config.py             # 설정
├── frontend/                  # 프론트엔드 (Streamlit)
├── tests/                     # 테스트
├── docs/                      # 문서
├── scripts/                   # 스크립트
├── .github/                   # GitHub Actions
└── docker-compose.yml          # 컨테이너 오케스트레이션
```

## 핵심 서비스 분석

### 1. 통합 서비스 (UnifiedService)
- 주식 데이터와 감성 분석을 통합
- 캐시 관리 및 데이터 품질 보증
- 병렬 처리로 성능 최적화

### 2. 주식 서비스 (StockService)
- Yahoo Finance API 연동
- 실시간 주식 데이터 수집
- 검색 및 필터링 기능

### 3. 감성 분석 서비스 (SentimentService)
- Reddit, Twitter 데이터 수집
- VADER 기반 감성 분석
- 트렌드 분석 및 예측

### 4. 캐시 관리자 (UnifiedCacheManager)
- 다계층 캐시 전략
- Redis 기반 분산 캐시
- 캐시 일관성 및 만료 관리
