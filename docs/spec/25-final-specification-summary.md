# InsiteChart 최종 스펙 문서 요약

## 1. 개요

### 1.1 프로젝트 배경

InsiteChart는 실시간 금융 데이터 분석과 소셜 미디어 감성 분석을 통합한 차세대 금융 분석 플랫폼입니다. 본 스펙 문서는 시스템의 전체 아키텍처, 기능 요구사항, 기술 구현 계획을 포함하는 포괄적인 기술 명세서입니다.

### 1.2 문서 구성

본 스펙 문서는 총 25개의 문서로 구성되며, 다음과 같은 주요 영역을 다룹니다:

1. **핵심 시스템 스펙** (문서 1-10)
2. **아키텍처 심화 스펙** (문서 11-17)
3. **품질 및 운영 스펙** (문서 18-25)

## 2. 핵심 시스템 스펙 요약

### 2.1 시스템 개요 (문서 1)

**목표**: 실시간 금융 데이터 분석 및 소셜 감성 분석 통합 플랫폼 구축

**핵심 기능**:
- 실시간 주식 데이터 모니터링
- 소셜 미디어 감성 분석
- 상관관계 분석 및 시각화
- 포트폴리오 관리
- 개인화된 대시보드

**기술 스택**:
- 프론트엔드: Angular/React, TypeScript
- 백엔드: Python FastAPI, Node.js
- 데이터베이스: PostgreSQL, Redis, MongoDB
- 메시징: Apache Kafka
- 컨테이너화: Docker, Kubernetes

### 2.2 시스템 아키텍처 (문서 2)

**아키텍처 패턴**: 마이크로서비스 아키텍처 (MSA)

**주요 구성 요소**:
- API 게이트웨이
- 인증/인가 서비스
- 데이터 수집 서비스
- 분석 엔진
- 감성 분석 서비스
- 알림 서비스
- 프론트엔드 애플리케이션

**데이터 흐름**:
```
외부 API → 데이터 수집 → 분석 엔진 → 감성 분석 → 데이터 저장 → 프론트엔드 전송
```

### 2.3 API 통합 (문서 3)

**외부 API 연동**:
- Yahoo Finance API: 실시간 주식 데이터
- Reddit API: 소셜 미디어 데이터
